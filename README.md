# Character Frequency Counter (C)

This C program counts the **frequency of each lowercase character** in a given string.

## Problem Statement
Given a string containing lowercase characters, print how many times each character appears.

## Approach
- Traverse the string character by character
- For each unvisited character, count its occurrences
- Mark visited characters using a special symbol (`#`)
- Print each character with its frequency

## Concepts Used
- Strings in C
- Nested loops
- Character arrays
- Basic frequency counting logic

## Sample Input
input:-google
output:-g-2
o-2
l-1
e-1
